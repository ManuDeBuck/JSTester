Javascript tester

Deze - hoogstwaarschijnlijk buggevulde - webapplicatie maakt het mogelijk om tijdens het werken met javascript toch nog een Python "doctest" gevoel te krijgen. Dien in het inputfield je code zoals hieronder getoond in:

Bron: dodona, Universiteit Gent, vak: Scriptingtalen, Prof. Dr. Peter Dawyndt

Klik vervolgens op de knop: GO, en copy-paste de resulterende code in je javascript code. Testen maar!

Voetnoot

De huidige versie ondersteunt ook het opgooien en controlen van AssertionErrors, het uitvoeren van eenvoudige functies en het vergelijken met het gewenste resultaat.
